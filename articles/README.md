# Articles

Rough drafts distilled from project conversations using the [distill-article](../skills/local/distill-article/) skill.

**Audience**: Senior developers, tech leads, AI/agentic practitioners, information systems researchers.

**How articles are produced**: After a substantive conversation in this project, the distill-article skill is run. It proposes article angles, discusses them with the human, and produces drafts for the approved angles.

**Status conventions**:
- `draft` — rough draft, needs human editing and voice
- `review` — structurally complete, being refined
- `published` — final version, with link to published location
